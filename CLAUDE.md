# symfony/ai-mate - Demo Project

## MCP Server Overview

This project demonstrates the **symfony/ai-mate** MCP server integration for PHP/Symfony development. The MCP server is named `mate` and provides tools and resources for AI-assisted development.

## Available MCP Tools

The `mate` MCP server provides these tools:

- `php-version` - Get the current PHP version
- `operating-system` - Get the current operating system
- `operating-system-family` - Get the OS family (unix, windows, etc.)
- `php-extensions` - List all loaded PHP extensions
- `test_tool` - Simple test tool for verification

## Available MCP Resources

Currently available resources:

- `test://info/about` - Information about the MCP server
- `test://info/help` - Help for using the MCP server

## Project Structure

The project uses the `.mate/` directory for configuration and extensions:

- `.mate/services.php` - Service configuration loaded into Symfony DI container
- `.mate/extensions.php` - Extension management (auto-generated by `mate discover`)
- `mate/` - Custom tools and resources (auto-discovered)
  - `mate/TestTool.php` - Example custom tool
  - `mate/TestResource.php` - Example custom resource

## Configuration

### Service Configuration (`.mate/services.php`)

Configure MCP server parameters and services:

```php
use Symfony\Component\DependencyInjection\Loader\Configurator\ContainerConfigurator;

return static function (ContainerConfigurator $container): void {
    $container->parameters()
        ->set('mate.cache_dir', sys_get_temp_dir().'/mate')
        ->set('mate.env_file', ['.env']) // Loads .mate/.env and .mate/.env.local
    ;
};
```

### Extension Management (`.mate/extensions.php`)

Enable/disable MCP extensions (managed by `mate discover`):

```php
return [
    'symfony/ai-mate' => ['enabled' => true],
];
```

## Development Guidelines

When working in this project:

1. **Custom Tools**: Add new tools in `mate/` directory - they're auto-discovered
2. **Configuration**: Use `.mate/services.php` for DI container configuration
3. **Extensions**: Run `mate discover` to scan for and register new extensions
4. **Testing**: Use the test tools to verify MCP server functionality
